# Student Management Database System

This project is a relational database designed to manage students, instructors, courses, enrollments, and grades.

## Features

- Student record management
- Course management
- Instructor management
- Enrollment tracking
- Grade tracking
- Multi-table JOIN queries

## Skills Demonstrated

- SQL
- Database Design
- Relational Databases
- Primary Keys
- Foreign Keys
- JOIN Queries
